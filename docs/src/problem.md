## 1. 为什么我的成绩出现在了导学阶段排行榜，而没有在基础阶段排行榜？

> 原因可能有两个：
>
> 1. 子模块命名不对
> 2. 没更新 exam-grading
>
> 请自行排查

## 2. 我已经完成了习题，并且添加了子模块，为什么没有成绩？

> 可能是你没有重新推送 exam-grading

## 3. 为什么我的习题无法 push 到远程仓库？

> 大概率是你忘记 fork 了，怎么 fork？回看导学阶段指导书的 Git 操作那一章，或自学。
